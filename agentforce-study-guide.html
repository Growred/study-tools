Plano de Estudos - 4 Semanas

Baseado no peso de cada dominio. Priorize os temas com maior peso primeiro. Marque cada item conforme concluir.

Semana 1 - AI Agents (35%) ~10-12h
Estudar como funciona o Reasoning Engine do Agentforce
Entender comportamento deterministico: filtros, variaveis, guardrails
Praticar configuracao de Standard e Custom Topics
Praticar configuracao de Standard e Custom Actions
Diferenciar Employee Agent, Service Agent e Sales Agent (SDR, Sales Coach)
Estudar seguranca: conceito de Agent User, permissoes, acesso a dados
Entender integracao com canais: Digital Experience, Email, Slack
Hands-on: Criar e configurar um agente no Trailhead Playground
Semana 2 - Prompt Engineering (20%) ~8-10h
Estudar quando e por que usar o Prompt Builder
Entender roles de usuario para gerenciar prompt templates
Dominar tipos: Field Generation e Flex Templates
Estudar tecnicas de grounding: merge fields, flows, Data Cloud, Apex
Aprender best practices para escrita de prompts eficazes
Hands-on: Criar prompt templates no Prompt Builder
Praticar o ciclo: criar, ativar, executar prompt templates
Semana 3 - Data Cloud & Deployment (40%) ~10-12h
Estudar Agentforce Data Library: conceito e tipos
Entender chunking e indexing de dados nao estruturados
Dominar Retrievers: Individual vs Ensemble
Diferenciar tipos de busca: Keyword, Vector e Hybrid
Estudar o Agentforce Testing Center: como testar agentes
Entender deploy: sandbox para producao (consideracoes e processo)
Estudar monitoramento de adocao e metricas de agentes
Hands-on: Configurar Data Library e testar agentes
Semana 4 - Multi-Agent + Revisao Geral ~8-10h
Estudar MCP (Model Context Protocol): proposito e casos de uso
Entender Agent-to-Agent Protocol e Agent API
Revisao geral: AI Agents (foco principal)
Revisao geral: Prompt Engineering
Revisao geral: Data Cloud e Deployment
Fazer simulados e practice exams
Revisar erros dos simulados e reforcar pontos fracos
Ultimo hands-on: percorrer cenarios completos end-to-end
